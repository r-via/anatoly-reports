# Anatoly Reports

Public audit reports generated by [Anatoly](https://github.com/r-via/anatoly) — Deep Audit Agent for codebases.

## Structure

```
{owner}--{repo}/
  {run-id}/
    report.md          # Executive summary, severity table, axis navigation
    axes/
      correction/      # Code correctness issues
      documentation/   # Documentation gaps
      best-practices/  # Code quality findings
      utility/         # Dead code and low-value code
      duplication/     # Duplicated code
      overengineering/ # Unnecessary complexity
      tests/           # Test coverage gaps
```

## Usage

Run `publish.sh` from the project you audited (where `.anatoly/` lives):

```bash
# Clone this repo somewhere
git clone https://github.com/r-via/anatoly-reports.git ~/anatoly-reports

# From the audited project directory:
~/anatoly-reports/publish.sh                # publish report + open issue
~/anatoly-reports/publish.sh --no-issue     # publish report only
~/anatoly-reports/publish.sh --dry          # preview without publishing
~/anatoly-reports/publish.sh --run <id>     # use a specific run
```

Requires `git` and `gh` (GitHub CLI).
