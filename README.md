# Anatoly Reports

Public audit reports generated by [Anatoly](https://github.com/r-via/anatoly) — Deep Audit Agent for TypeScript codebases.

## Structure

```
{owner}--{repo}/
  {run-id}/
    report.md          # Executive summary, severity table, axis navigation
    axes/
      correction/      # Code correctness issues
      documentation/   # Documentation gaps
      best-practices/  # Code quality findings
      utility/         # Dead code and low-value code
      duplication/     # Duplicated code
      overengineering/ # Unnecessary complexity
      tests/           # Test coverage gaps
```

## How it works

1. Fork a repository
2. Run `anatoly run` on the fork
3. Run `anatoly report upstream` to publish the report here and open an issue on the original repo

```bash
anatoly report upstream              # publish report + open issue
anatoly report upstream --no-issue   # publish report only
anatoly report upstream --dry        # preview without publishing
```
